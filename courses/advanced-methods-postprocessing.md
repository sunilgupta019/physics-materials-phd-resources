# Advanced Methods & Post-Processing

> Specialized methods that extend DFT/MD results.

| Method | Tool / Resource |
|---|---|
| Phonon calculations | phonopy — https://phonopy.github.io |
| Wannier functions | Wannier90 — https://www.wannier.org |
| Electronic transport | BoltzTraP2 |
| NEB / CI-NEB | VTST tools · LAMMPS NEB |
| DFT+U / GW / TDDFT | QE · VASP · ABINIT docs |
| Metadynamics / Umbrella sampling | PLUMED tutorials — https://www.plumed.org |
| Phase-field | MOOSE · FiPy · OpenFOAM |
| Kinetic Monte Carlo | KMC frameworks |

## Notes
These methods are typically layered on top of a converged DFT/MD setup. Pair each with a reproducible input set and version-controlled scripts (see [reproducibility](reproducibility-data-management.md)).
