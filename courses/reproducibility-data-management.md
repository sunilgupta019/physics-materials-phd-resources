# Reproducibility & Data Management

> Make your computational research reproducible and FAIR.

## Resources
- **The Carpentries** — https://carpentries.org (Software/Data Carpentry)
- **Zenodo** — https://zenodo.org (archive code/data, get a DOI)
- **figshare** — https://figshare.com

## Practices
- Git + GitHub workflows for research code (branches, PRs, releases)
- Docker for reproducible computational environments — https://training.docker.com
- Record exact input files, pseudopotentials, and software versions
- Archive datasets and tag releases with a DOI
