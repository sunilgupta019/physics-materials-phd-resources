# Contributing

Thanks for helping improve this PhD / research resource hub! 🙌

1. Open an **Issue** or **Pull Request** editing the relevant file in `courses/` or a top-level page.
2. Resources must be **free** (open or audit-free) and research-relevant.
3. Use the table format: `Resource | Provider | Format | Level | Notes`.

Legend: 🎥 video · 📄 notes/PDF · 📝 problem sets · 🧪 labs/sims · 🆓 fully free · 🏅 certificate.
